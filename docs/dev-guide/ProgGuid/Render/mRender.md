# Rendering

Rendering in LithTech is handled by engine code. A variety of exposed engine interfaces allow the game code to render primitives, objects, and world geometry.

This section contains the following rendering topics:

| #### ModelEdit Topics | #### Description |
| --- | --- |
| [Initialization ](Initial.md) | Describes how to initialize the rendering process. |
| [World Rendering ](World.md) | Describes how to prepare a world for rendering. |
| [Model Rendering ](Model.md) | Describes issues related to rendering models. |
| [Custom Rendering ](Custom.md) | Describes low-level interfaces for custom rending or specific effects. |
| [Render Styles ](Styles.md) | Describes a mechanism for consistent customization of rendering options. |
