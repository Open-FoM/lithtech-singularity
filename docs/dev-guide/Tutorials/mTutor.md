# Tutorials

This section lists all of the tutorials available for your use. The location of tutorials vary. Some tutorials are specific to tasks performed in ModelEdit, DEdit, or FXEd. Other tutorials combine tasks from mutiple tools, and may also include programming steps.

The following tutorials are available for the Jupiter system:

| #### Art and Level Tutorials | #### Description |
| --- | --- |
| [Adding an Attachment to Your Model ](TAddAtch.md) | Provides instruction on adding an attachment to your model in Jupiter. |
| [Adding Sky to Your Level ](../Dedit/TAddSky.md) | Provides instruction on adding a sky box. |
| [Creating Animations ](animat.md) | Provides instructions for programmatically playing simple model animations in multiplayer environment. |
| [Building Your First World ](../Dedit/TBldWrld.md) | Provides instruction on creating a simple world. |
| [Creating a Cut Scene ](cutscene.md) | Provides instruction on using ModelEdit, DEdit, and FXEd to create a cinematic cut scene in your game. |
| [Lighting Your World ](WrldLitn.md) | Provides instructions on selecting and applying different types of lighting. |
| [Using Brushes ](../Dedit/Brushes.md) | Provides instruction on using brushes. |
| [Working With AI ](../WAI/mAI.md) | Provides instructions on using the AI provided in the NOLF 2 game code. |



| #### Programming Tutorials | #### Description |
| --- | --- |
| [Multiplayer Flying Camera ](../ProgGuid/Prog/FlyCam.md) | Provides a tutorial on adding multiplayer capabilities to the [Sample Base ](../ProgGuid/Prog/SampleB.md)sample. |
| [Simple Physics ](../ProgGuid/Prog/SimpPhys.md) | Provides information about the problems and solutions that arise with physics in a multiplayer environment, and builds on the [Flying Camera ](../ProgGuid/Prog/FlyCam.md)tutorial as a base. |
| [Creating Pickups ](../ProgGuid/Prog/PickUps.md) | Provides instructions on creating objects in the game that characters can pick up and use. Builds on the [Simple Physics ](../ProgGuid/Prog/SimpPhys.html)tutorial. |
| [Creating Projectiles ](../ProgGuid/Prog/Bullets.md) | Provides instructions on creating moving objects in the game (such as bullets) that can connect and interact with models and world geometry. |
| [Creating Animations ](animat.md) | Provides instructions on creating simple model animations in a multi-player environment, and builds on the [Creating Projectiles ](../ProgGuid/Prog/Bullets.md)tutorial. |
| [Creating Animations 2 ](animat2.md) | Provides instructions on creating more advanced model animations in a multi-player environment and builds on the [Creating Animations ](animat.md)tutorial. |
| [Creating Doors Tutorial ](Doors1.md) | Provides instructions on creating a simple door that opens when touched by the player or a projectile. |
| [Creating Special Effects 1 Tutorial ](SP_FX1.md) | Provides instructions on creating special effects and combines the [Animations2 ](animat2.md)tutorial and the [clientfx ](../ProgGuid/ClientFX/mCliFX.md)tutorial. |
