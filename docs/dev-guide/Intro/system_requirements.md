## Windows Requirements

>

All Jupiter Editions

- Windows XP or Windows 2000 with SP2 or higher
- Microsoft Visual C++ .NET 2003 (7.1) or Microsoft Visual C++ .NET 2002 (7.0). Because of DirectX9, Microsoft Visual C++ 6.0 is no longer supported.
- Microsoft® DirectX® 9.0c SDK Retail Version
- Maya 6.0, 5.0, 4.5, 4.0, or 3.0
- 3D Studio Max 6.0 or 5.0 with Character Studio 4, 3D Studio Max 4.0 with Character Studio 3.1.2, or 3D Studio Max 3.0 with Character Studio 3.0.2. Character Studio 3.0, 3.0.1, 3.1, and 3.1.1 are not supported.
- Bink Video for Windows is optional for video playback. Bink is available from RAD Game Tools at [http://www.radgametools.com/ ](http://www.radgametools.com/)

Enterprise Edition Only

- SDKs for Maya 6.0, 5.0, 4.5, 4.0, or 3.0
- SDKs for 3D Studio Max 6.0 or 5.0 with Character Studio 4, 3D Studio Max 4.0 with Character Studio 3.1.2, or 3D Studio Max 3.0 with Character Studio 3.0.2.

## Required Environment Variables

>

In order to compile the engine, tools, and samples, you must create the following environment variables:

| **Environment Variable Name ** | **Value ** |
| --- | --- |
| GAME_MAIN_DIR | Install path to game executable directory (Ex. C:\TouchdownEntertainment\Jupiter_Ent\Development\TO2) |
| GAME_REZ_DIR | Install path to game resource directory (Ex. C:\TouchdownEntertainment\Jupiter_Ent\Development\TO2\Game) |
| GAME_TOOLS_DIR | Install path to Jupiter PC tools directory (Ex. C:\TouchdownEntertainment\Jupiter_Ent\Tools) |
