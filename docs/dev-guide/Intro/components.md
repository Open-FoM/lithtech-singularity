# Components

The various Jupiter components are located in the following subdirectories of \TouchdownEntertainment\Jupiter:

>
| \Development\TO2 | Binary No One Lives Forever™ 2 game and resources |
| --- | --- |
| \Docs | Documentation |
| \Engine\Built | Compiled engine libraries and executables |
| \Engine\ClientFx | ClientFx source code |
| \Engine\Libs | Engine libraries |
| \Engine\Runtime | Engine source code |
| \Engine\sdk | SDK header files |
| \Engine\Tools | Tool source code |
| \Game | No One Lives Forever™ 2 source code |
| \Libs | Library source code |
| \Samples | Sample source code |
| \Samples\bin | Compiled engine binaries |
| \Tools | Tool binaries |
